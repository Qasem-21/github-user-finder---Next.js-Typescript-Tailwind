# 🔍 GitHub User Search

A simple and modern web app built with **Next.js + TypeScript + React** that allows users to search for GitHub profiles and view their information.

---

## 🚀 Features

*  Search GitHub users by username
*  Display user profile (avatar, bio, followers, repos count)
*  Show user repositories
*  Loading state handling
*  Error handling (user not found, API errors)
*  Full responsive 

---

## 🛠 Tech Stack

* **Next.js**
* **React**
* **TypeScript**
* **TailwindCSS (basic styling)**

---

## 📡 API Used

GitHub REST API:

```
https://api.github.com/users/{username}
https://api.github.com/users/{username}/repos
```

## ⚙️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Qasem-21/github-user-finder---Next.js-Typescript-Tailwind
```

---

### 2. Install dependencies

```bash
npm install
```

---

### 3. Run the development server

```bash
npm run dev
```

---

### 4. Open in browser

```
http://localhost:3000
```
